# iQuiz Task

## Requirements

For development, you will only need Node.js and a node global package, installed in your environement.


### Installation Setup
     git clone https://github.com/cwmohit/iQuiz-Task.git    
     cd client
     npm install

### Run Testing server
  cd apis
  npm start

### Running the project
     npm start
     Visit to http://localhost:3000/
